# Steg

A game of steganography.  
CU Boulder Game Dev Club 2019/2020 collaborative project.  
MIT licensed.

Follow progress on [our Trello](https://trello.com/b/Z5izrS20/cugdc).

## Build locally

```bash
npm install
npm run watch
```

Then open `index.html` in your browser and start hacking.
